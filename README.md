# 关于实验三
	
 1. 在“四、Hadoop集群启动”处，需要先执行

    `$ exit`
    
	命令退出当前用户，到默认用户，然后执行
	
    `$ sudo service ssh start`

	然后再用

    `$ su - hadoop`

	切换回hadoop用户，执行

    `$ start-dfs.sh`

	后与实验指导一致。
